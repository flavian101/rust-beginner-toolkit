# Rust Installation & Hello World (Windows)

## 1. Install Rust

Rust uses an installer called **rustup**.

- Download the installer from the official site: [https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)
- Run the `.exe` file and follow the setup wizard (choose default installation).
- After installation, open **Command Prompt** or **PowerShell** and verify:

```powershell
rustc --version
```
