# Getting Started with Rust – A Beginner’s Toolkit

## Objective

Learn Rust basics, set up the environment, and run a simple Hello World program.

## Why Rust?

Rust is a modern systems programming language focused on safety and performance.  
Chosen for this project because it prevents memory bugs and is widely used in production.

## Prompt Plan

1. Install Rust and verify setup.
2. Write and run Hello World.
3. Identify common beginner errors.
4. Summarize official Rust docs.

## Deliverables

- Toolkit Document (Markdown/PDF)
- Working Codebase (Hello World in Rust)
